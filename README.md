# Customer_Responses_Analysis
Data Analytics Project
Overview

This project showcases an end-to-end data analytics workflow using Python, SQL, and Power BI. It focuses on transforming raw data into meaningful insights through data cleaning, analysis, visualization, and reporting, followed by clear presentation for stakeholders.

Dataset
Structured dataset containing business-related data
Includes multiple variables for analysis and insights
Data issues such as missing values and inconsistencies were handled during preprocessing

Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
MySQL Server (SQL queries and analysis)
Power BI (Interactive dashboard)
Gamma (Presentation creation)
Jupyter Notebook

Steps Performed
Loaded the dataset using Python
Conducted Exploratory Data Analysis (EDA)
Cleaned and preprocessed the data
Stored cleaned data in MySQL Server
Executed SQL queries for insights
Built an interactive Power BI dashboard
Created a detailed analytical report
Designed a presentation using Gamma

Dashboard
Interactive Power BI dashboard with key KPIs
Filters and visuals for trend and performance analysis
Designed for easy decision-making

Results & Insights
Identified key trends and patterns
Improved data clarity and usability
Delivered actionable insights through visual storytelling

How to Run the Project
Clone the repository
Install required Python libraries
Run the Jupyter Notebook for EDA and data cleaning
Import cleaned data into MySQL Server
Execute provided SQL queries
Open the Power BI file to view the dashboard

Review the report and presentation for final insights
